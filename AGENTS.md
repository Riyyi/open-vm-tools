### Triage labels

Uses the five canonical role strings (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`) as Status lines in issue files. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` at repo root + ADRs under `docs/adr/`. See `docs/agents/domain.md`.
